## git이란?

 1. 버전 관리 프로그램
 2. 터미널에서 git--version



 ## 초기 설치 시 git 세팅

 1. 터미널에 아래 코드를 순서대로 작성하기
 git config --global user.name '내이름'
 git config --global user.email '내 이메일'

 ## github에 코딩파일 업로드

 1. 내 프로필 repositories 클릭
 2. 우측상단 new 클릭
 3. Repository name에는 코딩파일 제목
 4. Creat Repository 버튼 클릭 

 5. VS code 좌측패널에서 git 아이콘 클릭
 6. Initialize Repository 클릭
 7. +눌러서 업로드할 파일만 Staged Changes로 이동
 8. Message입력칸에 어떤 것이 변경되었는지 입력하고 Commit 버튼 클릭
 
 9. 터미널에 순차적으로 복사 붙여넣기
 git remote add origin https://github.com/yeil1023/git-study.git
 git push -u origin main

 
 ## github에 코딩파일 수정 후 업로드

 1. 업로드할 파일만 + 눌러서 Stage Changes로 이동
 2. Message 입력칸에 어떤것이 변경되었는지 입력하고 Commit 버튼 클릭
 3. [Sync Changes 위화살표] 버튼 클릭하면 내 컴퓨터에서 github으로 코드 전송
